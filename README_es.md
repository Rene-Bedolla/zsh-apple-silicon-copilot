A diferencia de los dotfiles tradicionales, este repositorio está diseñado exclusivamente para **Macs con Apple Silicon**. Utiliza el entorno [MLX de Apple](https://github.com/ml-explore/mlx) para ejecutar un Copiloto de Desarrollo de IA Local directamente en tu máquina: sin llaves de API, sin suscripciones, 100% offline.

## ✨ Características Principales

- **🧠 IA Copilot Local (`explicar` y `git-ia`)**: Entiende comandos complejos y genera Commits Convencionales automáticamente usando modelos locales en español.
- **⚡ Instalación Zero-Friction**: Un solo comando para instalar Homebrew, Zsh, plugins y herramientas de CLI.
- **🛠️ Hacker Defaults (`macos-tweaks`)**: Menú interactivo para aplicar optimizaciones ocultas de velocidad en macOS.
- **📝 Captura Rápida Universal (`nota`)**: Guarda ideas al instante en formato markdown desde tu terminal.
- **🚀 CLI Moderno**: Reemplaza herramientas antiguas con `zoxide` (cd inteligente), `eza` (ls moderno), `bat` (clon de cat) y `fzf`.
