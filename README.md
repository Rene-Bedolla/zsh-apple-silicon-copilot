Unlike traditional dotfiles, this repository is designed exclusively for **Apple Silicon Macs**. It leverages the [Apple MLX framework](https://github.com/ml-explore/mlx) to run a Local AI Developer Copilot directly on your machine—no API keys, no subscriptions, 100% offline.

## ✨ Key Features

- **🧠 Local AI Copilot (`explicar` & `git-ia`)**: Understand terminal errors and generate Conventional Commits automatically using local MLX models.
- **⚡ Zero-Friction Setup**: One command to install Homebrew, Zsh, plugins, and CLI tools.
- **🛠️ macOS Hacker Defaults (`macos-tweaks`)**: An interactive menu to apply hidden macOS speed optimizations.
- **📝 Universal Quick Capture (`nota`)**: Instantly save markdown notes from your terminal.
- **🚀 Modern CLI Tools**: Replaces legacy tools with `zoxide` (smart cd), `eza` (modern ls), `bat` (cat clone), and `fzf`.
